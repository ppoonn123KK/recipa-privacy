# レシぱ プライバシーポリシー

最新更新日: 2024年10月27日

このプライバシーポリシー（以下「本ポリシー」といいます。）は、kazuki kagawa（以下「当方」といいます。）が提供するスマートフォン向けアプリケーション「レシぱ」（以下「本アプリ」といいます。）における利用者情報の取扱いについて定めるものです。

## 1. 収集する情報

本アプリは、サービスの提供および改善のために、以下の情報を収集することがあります。

- **利用者が入力する情報:**
  - 食材リスト、レシピ情報、料理履歴など、利用者が本アプリ内で作成・保存した情報。  
    これらの情報は、利用者のデバイス内にのみ保存されます。  
- **広告識別子:**
  - 本アプリでは、Google AdMobを利用して広告を表示しています。  
    これに伴い、パーソナライズド広告の表示を目的として、お使いのデバイスの広告ID（iOSの場合はIDFA、Androidの場合はAAID）を取得する場合があります。  
- **AIへの送信データ:**
  - 「AIレシピ提案」機能の利用時、利用者が管理する食材リストの「食材名」のみを、Google Gemini APIに送信します。  
  食材の数量やその他の個人を特定できる情報は送信されません。  

## 2. 情報の利用目的

収集した情報は、以下の目的で利用されます。  
- 本アプリの機能（在庫管理、レシピ提案、料理履歴の記録など）を提供するため。
- 利用者の興味・関心に合わせた広告を配信するため。
- サービスの利用状況を分析し、アプリの品質向上および不具合の修正に役立てるため。
- 利用者からのお問い合わせに対応するため。

## 3. 情報の第三者提供

本アプリは、以下の場合を除き、利用者の情報を第三者に提供することはありません。
- 法令に基づく場合。
- 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき。
- 上記「1. 収集する情報」に記載された、広告配信事業者（Google AdMob）およびAPI提供事業者（Google）への情報送信。

## 4. 情報の保護

本アプリは、収集した情報の紛失、盗用、悪用、不正アクセス、開示、改ざん、破壊から保護するために、合理的な技術的および管理的措置を講じています。

## 6. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービスの改善に伴い、事前の予告なく変更されることがあります。  
変更があった場合は、本アプリ内または適切な方法で利用者に通知します。
